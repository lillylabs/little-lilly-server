=== Firebase Server ===

Imports and saves instagram photos for each user.

* Little Lilly Test
  * Run using: `node index.js`
  * Database used: `https://little-lilly-test.firebaseio.com`
  * test.config.json needed 
* To run using Little Lilly
  * Run using: `FIREBASE_ENV=prod node index.js`
  * Database used: `https://little-lilly.firebaseio.com`
  * prod.config.json needed
